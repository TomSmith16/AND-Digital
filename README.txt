README for AND Digital Coding Challenge. 19/03/2019

Click the Filter button to filter the list of candidates by the desired skill, the skill can be entered in the text field to the right of the Filter button. The default skill entered is JavaScript as per the coding challenge requirements.

The skill can be searched for using any combination of letter cases. If the skill cannot be found you will be told no results have been found and you can try again.
